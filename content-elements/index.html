Content elements

Content elements are parts of content that exist inside blocks of text, such as buttons, headings and links.

Content elements

Alt text

Alt text is a way to label images. It's especially important for people who can’t see the images on our website and rely on screen readers to understand our content.

Alt text should describe the image in a brief sentence or two so that people who don’t see the image should come away with the same information as if they had.

[Example]

Buttons

Buttons should always contain actions. The language should be clear and concise. Capitalise the first word. Standard buttons on our site include:

  • Sign the petition
  • Subscribe
  • Donate
  • Read more

Headings

Typography and hierarchy is important. It’s a vital element of our design. H1 and h2 tags are used by search engines to understand the structure of our content. Using headings also helps make our content readable and accessible by breaking up text and making it easily scannable.

Formatting

Each page should have only one h1, the page title. The first level of subheadings are h2s. We also sometimes use h3s, though we avoid using too many different levels. We never use bold, italics or underlining to style text as a heading.

Style

Headings should be in sentence case. This aids readability and fits with our voice.

Substance

Text in headings should be concise and informative.

Links

Provide a link whenever you’re referring to something on an external website. Use links to point users to relevant content and trusted external resources.

Good links tell people where they’re going to end up when they click on them. Avoid using words that wouldn’t make sense without the link. And definitely don’t use “click here” or “read this”. Write the sentence as you normally would, and link relevant keywords.

Don’t include preceding articles (a, an, the, our) when you link text. For example:

  • Yes: Read the annual report for more on ActionAid's impact.
  • No: Read the annual report for more on ActionAid's impact.

If a link comes at the end of a sentence or before a comma, don’t link the punctuation mark.
