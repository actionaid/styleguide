Content style guide

A guide to help us write clear and consistent content. Please use it as a reference when you’re writing for ActionAid.

This guide goes beyond grammar and style, also defining how we express ourselves as an organisation.

Under Content model in the menu, there are sections for each of the types of content we publish, so you can start there, or go straight to a specific style entry.

The entire guide is searchable, so you can go straight to the item you’re looking for.

Where something isn’t specified here, we follow the Guardian style guide. Where something isn’t explicitly covered in the Guardian style guide, we follow Guardian usage.

As an organisation working for a more equal world, we are especially careful to use inclusive language. See the section on inclusivity for more details.
